# React View Transitions API demo

mozilla のデモサイトを React で書き直しています。

## View Transitions API とは

簡単に DOM を 変更しながら、アニメーション遷移を作成できます。

## 技術スタック

- React 18
- TypeScript
- Vite（ビルドツール）
- Vercel(デプロイ先)
